# 🚀 PC Toolkit Pro v2.1 - Major Feature Update

**Release Date:** December 15, 2025 | **Version:** 2.1.0 | **Build:** 2025.12.15

---

## 🎯 Release Highlights

This major update introduces **6 powerful new features**, significant **performance improvements**, and a completely **redesigned user interface** with enhanced theming capabilities. PC Toolkit Pro v2.1 delivers the most comprehensive Windows system management experience yet.

---

## ✨ New Features

### 🎨 **Dynamic Theme System**
- **Light/Dark Theme Toggle** - Seamless switching between modern light and dark themes
- **System Theme Detection** - Automatically matches Windows theme preferences
- **Custom Color Schemes** - Personalize accent colors and UI elements
- **High Contrast Support** - Enhanced accessibility for visually impaired users

### 📈 **Advanced GPU Monitoring**
- **Real-time GPU Usage** - Live tracking of GPU utilization across all graphics cards
- **Temperature Monitoring** - GPU temperature alerts and thermal throttling detection
- **Memory Usage Tracking** - VRAM consumption monitoring for gaming and professional workloads
- **Multi-GPU Support** - Comprehensive monitoring for systems with multiple graphics cards

### 🗂️ **Intelligent Registry Cleaner**
- **Safe Registry Optimization** - AI-powered registry cleaning with automatic safety checks
- **Automatic Backups** - Complete registry backups before any modifications
- **Selective Cleaning** - Choose specific registry categories for targeted optimization
- **Rollback Protection** - One-click restore functionality for peace of mind

### 🔄 **Smart Auto-Update System**
- **Background Updates** - Seamless updates without interrupting workflow
- **Rollback Support** - Instant rollback to previous versions if issues occur
- **Update Scheduling** - Configure automatic update times and frequency
- **Bandwidth Control** - Limit update download speeds to preserve network performance

### 📱 **Enhanced Notification System**
- **Toast Notifications** - Modern Windows 10/11 style notifications
- **Operation Status** - Real-time updates on cleaning and monitoring operations
- **Custom Alerts** - Configurable alerts for system thresholds and events
- **Silent Mode** - Disable notifications during presentations or gaming

### 🔍 **Advanced Duplicate File Finder**
- **Smart Detection Algorithm** - AI-powered duplicate detection with 99.9% accuracy
- **Multiple Scan Methods** - Hash-based, size-based, and content-based scanning
- **Preview Integration** - Built-in file preview for safe duplicate removal
- **Batch Operations** - Select and remove multiple duplicates simultaneously

## 🔧 Technical Details

### **System Requirements**
- **Minimum:** Windows 10 (Build 19041) or Windows 11
- **Recommended:** Windows 11 with 8GB RAM and SSD storage
- **Python:** 3.8+ (for source installation)
- **Dependencies:** PyQt6 6.9.1+, psutil 7.0.0+, humanize 4.12.3+

### **Installation Size**
- **Executable:** 45MB (compressed), 120MB (installed)
- **Source Code:** 15MB (with dependencies)
- **Runtime Memory:** 80-150MB (depending on active features)

### **Compatibility**
- **Architecture:** x64, ARM64 (Windows 11)
- **Virtualization:** VMware, Hyper-V, VirtualBox compatible
- **Antivirus:** Tested with Windows Defender, Norton, McAfee, Bitdefender
